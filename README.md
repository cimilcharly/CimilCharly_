## ✨ Features

- 🎨 **Dark workspace design** with glassmorphism and micro-animations
- ⌨️ **Command Palette** (`⌘K`) for quick navigation
- 📱 **Fully responsive** across all screen sizes
- 🔍 **SEO optimized** — JSON-LD schema, sitemap, Open Graph, Google Search Console verified
- ⚡ **SSR via Nitro + Vercel** for fast page loads
- 📄 **Resume download** built-in

## 🏃 Run Locally

```bash
# Clone the repo
git clone https://github.com/cimilcharly/CimilCharly_.git
cd CimilCharly_

# Install dependencies
npm install

# Start dev server
npm run dev
```

Open [http://localhost:8080](http://localhost:8080) in your browser.
